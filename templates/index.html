Work at Olist - Test Implementation

The project is an API responsible for:

  • Register phone starts and ends calls
  • Generate monthly bill

Description

A REST API that receives call records and generate monthly bills for a given telephone number. This fork is a test resolution for a job application at Olist as Python Web Developer.

Deployed instance

The project was deployed on Heroku.
